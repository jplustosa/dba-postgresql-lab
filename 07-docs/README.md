# PostgreSQL Lab — DBA

## Objetivo
Simular administração básica de PostgreSQL.

## Ambiente
- PostgreSQL 15
- Docker

## Atividades
- Setup do banco
- Criação de schema
- Inserção de dados
- Análise de query lenta
- Criação de índice
- Backup e restore 

## Resultados
- Query otimizada com índice
- Redução significativa no tempo
- Restore validado

## Aprendizados
- Índices impactam diretamente performance
- Backup sem restore não é confiável
